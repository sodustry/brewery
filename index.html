ABOUT THIS APP
思考の醸造とは？
コンセプト
脳を「倉庫」から「工場」へ
現代人は情報過多により、常に頭のメモリがパンクしている。思いついてもすぐ忘れたり、「早く形にしなければ」と焦ったりしがちである。

本アプリの目的は、「安心して、きれいに忘れること」。断片的なアイデアを一度完全に預けて脳に余白を作り、システムが裏側でじっくりと発酵させ、本当に価値のあるものだけを昇華させる。
裏側で動く4つのアルゴリズム
『思考の整理学』（外山滋比古）の思想をデジタルロジックに翻訳
ALGORITHM 01
🫧
醗酵アルゴリズム
メモを保存した瞬間、テキストが画面から消える。これは脳に「もう忘れていい」と錯覚させるUI演出。システムは裏側でタイムスタンプを記録し、設定した熟成期間、意図的にメモを隠す（寝かせる）。
→ ファストメモ画面で体験
ALGORITHM 02
🌅
朝飯前アルゴリズム
前回のアクセスから8時間以上の空白を検知した初回起動時にのみ、熟成済みメモを提示する。脳がリセットされた状態のときだけ判断させることで、直感の精度を最大化する。
→ デバッグ「朝レビュー」で体験
ALGORITHM 03
🌫
忘却アルゴリズム
「ピンとこない」と判定されたメモ、または30日以上放置されたメモは、削除確認のダイアログなしに自動で地下倉庫へ沈む。人間が睡眠中に不要な記憶を消去するように、アプリが忘却を代行する。
→ デバッグ「フェード実行」で体験
ALGORITHM 04
⚗️
カクテル＆触媒アルゴリズム
メインテーマのキーワード（名詞など）を抽出し、過去のメモから「一見無関係だが抽象概念がどこか重なるメモ（弱いつながり）」を1件スパイスとして提示。異質な2要素の掛け合わせが、セレンディピティ（偶然の発見）を生む。
→ ⚗️触媒タブで体験
